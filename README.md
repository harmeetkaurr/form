
## install virtual environment

## Download Django
```
$ pip install django
```

## Download Crispy-Form
```
$ pip install django-crispy-forms
```
Then 
```
$ python manage.py makemigrations
$ python manage.py migrate
```
## Start the app
```
$ python manage.py runserver
```

Open browser, <http://localhost:8000>.
## Stop the app

           Ctrl+C
